Methodology

5-layer Bayesian pipeline — each layer refines the attack/defense ratings used in the Poisson goal model (λ = BASE × atk × def):
1
Live Elo ratings — base priors from eloratings.net (244 teams, updated monthly). Spain 2129, Argentina 2128, Qatar 1437. Converted to atk/def via a linear formula calibrated against WC 2022 betting markets.
2
Squad / pedigree — StatsBomb xG data from WC 2022 (top-5 player xG quality), WC pedigree points (wins/titles), and manager score (0–10) combine into a ±10% multiplier.
3
392 qualifying matches — all 6 confederations (ESPN public API), Bayesian update at lr=0.04. Includes non-qualifiers so e.g. Argentina vs Bolivia still informs Argentina's rating.
4
WC 2026 results — live match data from ESPN, Bayesian update at lr=0.08. Penalties discounted 40%, own goals discounted 100%. Red-card matches scaled to 35–70%.
5
Form + injuries (AI) — Tavily web search (1 query/team) feeds Groq LLaMA 3.3 70B, which scores both form (0–10) and injury severity (0–10). Combined into a final ×0.90–1.04 multiplier per team.
Monte Carlo — 50,000 independent tournament simulations. Each: simulate remaining group matches (Poisson draws), select top-2 + best 8 third-place teams, run a 32-team knockout bracket. Championship probability = fraction of sims won.

Limitations — exact FIFA third-place bracket seeding simplified to random draw. Probabilities reflect model uncertainty, not betting advice.
