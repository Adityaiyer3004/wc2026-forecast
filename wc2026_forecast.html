Methodology

5-layer Bayesian pipeline — each layer refines attack/defense ratings used in the Poisson goal model (λ = BASE × atk × def):
1
Live Elo ratings — base priors from eloratings.net (244 teams). Converted to atk/def via formula calibrated against WC 2022 betting markets.
2
Squad / pedigree — StatsBomb xG (WC 2022), WC pedigree points, and manager score (0–10) combine into a ±10% multiplier.
3
392 qualifying matches — all 6 confederations via ESPN, Bayesian update lr=0.04. Non-qualifiers included so Bolivia vs Argentina still informs Argentina's rating.
4
WC 2026 results — live from ESPN, lr=0.08. Penalties −40%, own goals −100%.
5
Form + injuries (AI) — Tavily web search → Groq LLaMA 3.3 70B scores form (0–10) and injury severity (0–10). Combined multiplier range 0.90–1.04.
Monte Carlo — 50,000 simulations. Each: simulate remaining group matches (Poisson), advance top-2 + best 8 third-place to 32-team knockout, run bracket.

Validated against WC 2022 — Argentina ranked #3 pre-tournament (11.8%), correctly dropped to 3.4% after the Saudi upset, recovered to 49.8% in the Final. Brier score beat the random baseline at every checkpoint.

Limitations — third-place bracket seeding simplified to random draw. Analysis only — not betting advice.
